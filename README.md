🪨📄✂️ Rock-Paper-Scissors-Lizard-Spock 
Классическая игра с расширенными правилами (камень, ножницы, бумага, ящерица, Спок).
Сражайтесь против компьютера с разными уровнями сложности, отслеживайте статистику, устанавливайте рекорды и наслаждайтесь цветным интерфейсом.
Реализована на 7 языках программирования с единым набором возможностей.

🚀 Возможности
5 вариантов хода – камень 🪨, ножницы ✂️, бумага 📄, ящерица 🦎, Спок 🖖 (по мотивам «Теории большого взрыва»).

Три уровня сложности:

Easy – случайные ходы компьютера.

Medium – компьютер учитывает ваши предыдущие ходы (адаптивная стратегия).

Hard – компьютер анализирует паттерны и пытается предугадать ваш следующий ход.

Серии раундов – можно сыграть один раунд или серию (до 3, 5 или 7 побед).

Статистика – количество игр, побед, поражений, ничьих, процент побед.

Сохранение рекордов – лучшие результаты хранятся в локальном файле.

Цветной вывод – ходы, результаты и сообщения выделены для лучшей читаемости.

Интерактивный выбор – меню для выбора режима, сложности, количества раундов.

📖 Использование
Синтаксис (единый для всех версий):

bash
<команда> [опции]
Опции
Опция	Описание
-m, --mode <classic|extended>	Режим: classic (3 варианта) или extended (5 вариантов, по умолчанию)
-d, --difficulty <easy|medium|hard>	Уровень сложности (по умолчанию medium)
-r, --rounds <N>	Играть до N побед (по умолчанию 1)
-s, --stats	Показать статистику и выйти
-h, --help	Справка
Примеры
bash
# Расширенный режим (5 вариантов), средняя сложность, 1 раунд
python rps.py

# Классический режим, лёгкая сложность, до 3 побед
python rps.py -m classic -d easy -r 3

# Показать статистику
python rps.py -s
Управление
В игре вводите номер варианта (1–5) или букву (r, p, s, l, k).

q – выход во время игры.

После каждого раунда программа показывает результат и предлагает продолжить.

🧠 Логика и алгоритмы
Easy – компьютер выбирает случайный ход из доступных.

Medium – компьютер хранит историю ваших ходов и выбирает ход, который побеждает ваш самый частый предыдущий ход.

Hard – компьютер анализирует последовательности из двух ваших последних ходов и предсказывает следующий, выбирая контрход.

Используются классические правила:

Камень побеждает ножницы и ящерицу.

Ножницы побеждают бумагу и ящерицу.

Бумага побеждает камень и Спока.

Ящерица побеждает бумагу и Спока.

Спок побеждает камень и ножницы.

✨ Дополнительные фичи
Автосохранение – статистика записывается в файл после каждой игры.

Цветной вывод – ANSI‑коды для выделения ходов и результатов.

Гибкая настройка – выбор режима, сложности и количества раундов через аргументы командной строки.
